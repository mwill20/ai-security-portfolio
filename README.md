### For Deployment

This portfolio is deployed on [Render](https://render.com/). To deploy your own version:

1. Fork this repository
2. Create a new Web Service on Render and connect your GitHub repository
3. Use the following build command:
   ```bash
   npm install -g serve
   serve -s .
   ```
4. Set the publish directory to the root of the repository
