Project Overview

SageVault is a security-minded RAG (Retrieval-Augmented Generation) application designed for secure document and codebase analysis. It provides AI-powered Q&A with built-in security guardrails, ensuring safe exploration of sensitive information while maintaining strict access controls and data protection measures.

Security First: Built with multiple layers of protection including allowlist/blocklist controls, session-only key storage, and injection-aware prompts.

Security Features

Content Security

Advanced filtering with allowlist/blocklist for file types and content, preventing processing of potentially malicious or sensitive files.

Data Protection

Session-only API key storage with no persistent logging of sensitive information or queries.

Injection Protection

Built-in guardrails to detect and prevent prompt injection attacks against the AI models.

Controlled Access

Single source per session policy ensures clean separation between different document sets and repositories.

Technical Implementation

SageVault is built with enterprise-grade security in mind, featuring:

  • Multi-LLM support (OpenAI, Anthropic, Google Gemini, Groq) with secure API key management
  • Chroma vector store with configurable chunking and overlap for optimal information retrieval
  • MMR (Maximal Marginal Relevance) re-ranking for diverse and relevant results
  • Comprehensive analytics for monitoring usage patterns and potential security events
  • Export capabilities with proper data sanitization
